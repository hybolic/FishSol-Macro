  fishSol Macro

A macro for the Roblox game Sol's RNG, specifically to automate its fishing minigame.

Installation

  1. Download AutoHotKey v1.1 (not v2.0) and install it
  2. Download the latest release of fishSol, and extract the .zip file into an empty folder
  3. For first time launch please use LaunchMacro.ahk as this builds the plugin manager for use with the macro. for debug purposes the prebuilt version with all plugins is included by default
  4. Run fishSol through the .ahk file by double-clicking it, or right-clicking it and selecting "Run script"

if a plugin is missing just relaunch the LaunchMacro.ahk to rebuild the PluginManager and hopefully fix the issue

Plugins

Download your desired plugin from the release or from the plugins folder in the source code, and drag it into the plugins folder. A tab for the plugin should automatically show up when you restart the macro.

Features

  • Auto-Fish
  • Auto-Sell
  • Different Resolution Settings
  • Different Pathing Options
  • QWERTY and AZERTY keyboard support
  • Auto-Rejoin Failsafe
  • Fishing & Pathing Failsafes
  • Auto-Unequip
  • More to come!
